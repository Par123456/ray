## 💎 ویژگی‌های برتر و متمایز نسبت به سایر کلاینت‌ها (v2rayNG)

برنامه **AnishtayiN-RAY** مجهز به چندین قابلیت ممتاز و انحصاری است که تجربه وب‌گردی آزاد و پایدار را بهبود می‌بخشد:

| ویژگی (Feature) | آلیشتایین-رِی (AnishtayiN-RAY) | ویتوری (v2rayNG) | توضیحات |
| :--- | :---: | :---: | :--- |
| **بهبود هوشمند تاخیر (Latency Auto-Healer)** | **✓ (فعال خودکار)** | ✗ | پایش پیوسته پینگ در پس‌زمینه و سوئیچ آنی به سریع‌ترین سرور در زمان لود ضعیف |
| **نمایش موقعیت و کارور خروجی (IP Geo-ISP)** | **✓ (کامل با پرچم کشور)** | ✗ | نمایش آی‌پی واقعی خروجی، شرکت ارائه‌دهنده سرور (ISP) و کشور مبدا |
| **اتصال مستقیم تلگرام (Direct Telegram Proxy)** | **✓ (بدون نیاز به پورت کمکی)** | ✗ | ساخت و تنظیم همزمان پروکسی لوکال SOCKS5 برای باز کردن تلگرام با فشردن یک دکمه |
| **طراحی بومی و مدرن (Jetpack Compose M3)** | **✓ (کاملا بومی)** | ✗/تا حدودی | رابط کاربری پیشرو، پویانمایی‌های روان و تم کاملا تیره برای محافظت از چشم |

## 🚀 قابلیت‌های کلیدی / Key Features

- **پشتیبانی کامل از پروتکل‌های مطرح:**
  - `Shadowsocks`  (ایمن و سبک)
  - `VLESS` (واقعیت نسل جدید Reality / TLS)
  - `VMess` (شناور و منعطف)
  - `Trojan` (تونل همرنگ وب‌سایت‌های عادی)
- **رمزگشایی هوشمند لینک‌ها:** اتصال پرسرعت از طریق کپی کردن مستقیم کانفیگ از حافظه موقت (Clipboard) یا اسکن کدهای سریع QR.
- **ترافیک دقیق پس‌زمینه:** نظارت لحظه‌ای بر پهنای باند ورودی و خروجی با استفاده از آمارهای سیستمی اندروید (`TrafficStats`).
- **جهت‌دهی ترافیک (Routing Modes):**
  - از فیلتر گذراندن همه چیز (Global)
  - بای‌پَس کردن سایت‌های ایرانی و محلی (Bypass Iran & LAN)‌ جهت مصرف نیم‌بهای اینترنت داخلی.
  - عبور مستقیم سایت‌های انتخابی (Only Direct).

## 🛠️ نصب و راه‌اندازی برای برنامه‌نویسان / Technical Setup

### پیش‌نیازها
- اندروید استودیو (Android Studio Ladybug یا جدیدتر)
- جاوا نسخه ۱۷ (JDK 17)
- گوشی اندروید با نسخه Android 8.0 (Oreo) یا بالاتر

### ساخت و اجرا
```bash
# کلون کردن پروژه
git clone https://github.com/AnishtayiN/ray.git

# ورود به دایرکتوری برنامه
cd ray

# پکیج نیم پیش‌فرض
# com.AnishtayiN.ray
```
